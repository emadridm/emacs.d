## Vale

Vale (https://vale.sh/) is an open source, command-line tool that brings your editorial style guide to life. Install Vale with package manager or download from Releases and place in your $PATH. Then, generate your configuration using https://vale.sh/generator and copy this in the $HOME/.vale.ini file.
